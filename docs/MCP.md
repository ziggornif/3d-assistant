## What is MCP?

MCP (Model Context Protocol) is a protocol that enables AI models and automation tools to interact with applications programmatically. This integration allows generating 3D print quotes without using the web interface.

## Available Tools

1. **list_materials** - List available printing materials with prices
2. **upload_model** - Upload 3D models (STL/3MF) from base64-encoded data
3. **configure_model** - Configure model with material and quantity
4. **generate_quote** - Generate complete quote for session
